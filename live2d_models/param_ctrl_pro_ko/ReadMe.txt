    샘플 모델
    파라미터 컨트롤러용 샘플

============================================================

    애니메이션의 제작 효율이 오르는 「파라미터 컨트롤러」를 사용한 샘플입니다.
    애니메이션 데이터에 "파라미터 컨트롤러"를 추가했습니다.
    파라미터 컨트롤러의 타겟 추종을 사용하는 것으로, 무언가를 가지거나, 지면에 접하거나 등, 지금까지 작성이 어려웠던 애니메이션을 효율적으로 작성할 수 있게 됩니다.
    추종하는 움직임을 베이크하여 임베디드 모션 데이터에 반영할 수도 있습니다.
